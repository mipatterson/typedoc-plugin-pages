Optional replaceGlobalsPage

replaceGlobalsPage: boolean
  • Defined in src/options/models/plugin-options.ts:70

Whether or not to replace the "Globals" page with a "Home" page

The effect of this is that the "Globals" link in the header breadcrumbs will be replaced with a "Home" page that links directly to the root README page and the Globals link in the navigation sidebar will be hidden.

This defaults to false.
